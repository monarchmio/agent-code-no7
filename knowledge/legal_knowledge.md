# 法律知识库配置说明

## 知识库概述

Agent Code No.7 法律知识库采用 RAG（检索增强生成）架构，为法律相关特工提供精准的法规、案例和实务知识支持。

## 知识库分类

### 1. 法律法规库 (chinese_laws_db)
- **内容**：全国人大立法、国务院行政法规、部门规章、地方性法规
- **更新频率**：每周自动同步
- **数据来源**：国家法律法规数据库、北大法宝
- **向量化方式**：按条款粒度切分，保留法规元数据（名称、文号、生效日期、效力状态）

### 2. 司法案例库 (court_cases_db)
- **内容**：最高法指导案例、各级法院公报案例、典型案例
- **更新频率**：每日增量更新
- **数据来源**：中国裁判文书网、最高法案例数据库
- **向量化方式**：按裁判要旨和案件事实分别索引

### 3. 合同模板库 (contract_templates)
- **内容**：各类标准合同模板（采购、服务、劳动、投资、保密等）
- **更新频率**：季度更新
- **标注内容**：必备条款清单、常见风险点、行业惯例

### 4. 法律解释库 (legal_interpretations)
- **内容**：司法解释、部门答复、行业监管FAQ
- **向量化方式**：按问答对形式索引

## 检索策略

```json
{
  "retrieval_config": {
    "top_k": 10,
    "similarity_threshold": 0.75,
    "reranking": true,
    "reranker_model": "bge-reranker-v2",
    "hybrid_search": {
      "dense_weight": 0.7,
      "sparse_weight": 0.3
    }
  }
}
```

## 注意事项

- 法规检索结果需验证时效性（是否现行有效）
- 案例引用需区分指导案例（具有参照效力）和普通案例
- 不同法域的法规需明确标注适用范围
