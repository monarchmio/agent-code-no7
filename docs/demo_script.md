## 录制注意事项

1. **提前准备好演示数据**（示例合同、公司名等），避免使用真实敏感数据
2. **网络环境**：确保网络稳定，避免加载过慢
3. **字体大小**：放大到 125%-150%，确保录屏清晰
4. **背景音乐**：选择轻快的科技感BGM（注意版权）
5. **字幕**：建议添加中文字幕，提升可看性
