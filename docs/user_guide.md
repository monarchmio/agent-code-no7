# Agent Code No.7 — 使用指南

## 快速入门

### 第一步：选择特工或工作流

根据你的任务类型，选择合适的特工或工作流：

| 我想要... | 推荐方式 |
|-----------|---------|
| 审查一份合同 | 🔄 合同审查工作流 |
| 研究一个法律问题 | 🔄 法律意见书工作流 / ⚖️ Agent-02 |
| 分析财务报表 | 🔄 审计分析工作流 / 🦅 Agent-03 |
| 做行业研究 | 🔄 行业研究工作流 / 🔭 Agent-05 |
| 做尽职调查 | 🔄 尽调工作流 |
| 税务筹划建议 | 🏦 Agent-04 |
| 不确定找谁 | 🕵️ Agent-07 总指挥（他会帮你分配） |

### 第二步：输入任务

- **直接对话**：用自然语言描述你的需求
- **上传文件**：支持 PDF、DOCX、XLSX 格式
- **使用模板**：参考 `prompts/` 目录下的提示词模板获得更好效果

### 第三步：获取结果

- 特工会逐步展示分析过程
- 最终输出结构化报告
- 可以追问或要求修改

## 使用示例

### 示例 1：合同审查

```
用户：请帮我审查这份技术服务合同，重点关注知识产权归属和违约责任条款。
[上传合同.pdf]

Agent-07：收到，我已将任务分配给合同卫士（Agent-01）和法律猎手（Agent-02）。
- Agent-01 正在解析合同条款...
- Agent-02 正在检索相关法规...

[2分钟后]

Agent-07：审查完成，以下是报告摘要：
📊 综合风险评分：68/100
🔴 高风险条款 3 项
🟡 中风险条款 5 项
[查看完整报告]
```

### 示例 2：多特工协作尽调

```
用户：我们准备投资 XX 科技公司，请做一个初步的尽职调查。

Agent-07：了解，我将调度以下特工并行工作：
✅ Agent-06 尽调特工 → 工商信息与关联关系
✅ Agent-02 法律猎手 → 诉讼与合规记录
✅ Agent-03 审计鹰眼 → 财务健康度分析
✅ Agent-05 研究探员 → 行业地位与竞争力

[5分钟后]

Agent-07：尽调完成，综合评级 🟡 中等风险。
发现 2 项重大关注事项，详见报告...
```

## 最佳实践

1. **提供充足上下文**：描述越详细，特工输出越精准
2. **善用总指挥**：不确定找谁时，直接问 Agent-07
3. **迭代优化**：对结果不满意可以追问或要求重新分析
4. **组合使用**：复杂任务使用工作流比单个特工效果更好
5. **注意免责**：所有输出仅供参考，重大决策请咨询专业人士
