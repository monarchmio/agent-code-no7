# Agent Code No.7 — AI特工局

> 🕵️ 每一次任务，使命必达

**Agent Code No.7** 是基于 ClawPro (OpenClaw) 平台构建的多智能体协作系统，以"AI特工局"为核心概念，专为律所、会计师事务所、咨询公司打造一站式智能办公解决方案。

## 特工阵容

- 🎯 **总指挥 Commander** — 任务编排、特工调度、质量把控
- 📜 **合同卫士 Contract Guardian** — 合同审查、风险识别、修改建议
- ⚖️ **法律猎手 Legal Hunter** — 法规检索、案例分析、法律意见
- 🦅 **审计鹰眼 Audit Eagle** — 财报分析、异常检测、审计底稿
- 🏦 **税务管家 Tax Butler** — 税务筹划、合规审查、政策匹配
- 🔭 **研究探员 Research Scout** — 行业研究、竞品分析、市场洞察
- 🔍 **尽调特工 Due Diligence** — 尽职调查、背景核查、风险评级

## 核心能力

- **智能编排**：总指挥根据需求自动拆解任务、分发给最合适的特工
- **并行协作**：多特工同时工作，5分钟完成传统数天的任务
- **深度专业**：每位特工深耕一个领域，Prompt + 知识库深度优化
- **即插即用**：SKILL.md 标准格式，导入 ClawPro 即可使用

## 典型场景

| 场景 | 效率提升 |
|------|---------|
| 合同审查 | 4-8小时 → 2分钟 |
| 尽职调查 | 1-3个月 → 5分钟 |
| 行研报告 | 1-2周 → 5分钟 |
| 法律意见 | 1-2天 → 3分钟 |
